Frame für das HTML-Element des Bildes anzeigen.
Kann veranschaulichen, ob das Bild selbst komplett mit dem umgebenden Element skaliert. Mithilfe dieses Flags kann die Bearbeitung/Prüfung des übergebenen luxImgSrc deaktiviert werden.
Dadurch ist es z.B. möglich, relative Links (siehe letzte Option in dem Select-Feld) darzustellen.
Hinweis: Weil es kein Backend gibt, wird das Bild hier nicht korrekt dargestellt.Ein Blick in die Developer-Tools zeigt aber eine theoretisch korrekte Image-Source an.
