Ein LuxIconComponent in dem Tile anzeigen.
Wenn 'false' wird stattdessen in dem Beispiel ein LuxImageComponent genutzt. Diese Property aktiviert eine Notifizierungsansicht an der Tile.
Wenn 'undefined', wird nichts angezeigt. Bei 'true' wird ein blauer Kreis dargestellt und bei 'false' ein ausgegrauter Kreis. Die Property definiert den aktuellen Zähler für die Tile.
Wenn der Wert den luxCounterCap überschreitet, wird die Zahl abgekürzt. Diese Property bestimmt die Obergrenze für luxCounter.
