Mit der Version 13.0.0 wurde bei den Lux-Form-Controls das Margin entfernt. Damit kann und muss der Abstand in einem Formular individuell festgelegt werden.
Für die Lux-Layout-Form-Row, lässt sich das über das Attribut luxGap regeln.
Wichtig ist es auch den Abstand zwischen zwei Reihen zu definieren, damit die Lesbarkeit der Formulare erhalten bleibt.

In einer Layout-Form-Row können neben den Formularelementen auch Buttons eingefügt werden. Diese werden neben den Default-Ac Komponenten zentriert angezeigt. Werden mehrere Buttons eingefügt, sollten diese in einem div platziert werden. Dieses div ist dann das eigentliche Row-Item und kann zusätzlich über eine individuelle Klasse die Abstände zwischen den Buttons regeln.

Haben die Formular-Elemente in einer Reihe das Attribut luxNoTopLabel oder luxNoBottomLabel gesetzt, muss dem Button über die Directive luxLayoutRowItem ebenfalls ein entsprechendes Attribut übergeben werden. Diese Direktive greift auch bei einem div, dass als Button-Container dient.
