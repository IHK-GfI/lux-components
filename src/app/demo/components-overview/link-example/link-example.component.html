Bestimmt die URL, zu der der Link routet.
Wenn die URL nicht mit http beginnt, wird automatisch der Angular-Router für Routing innerhalb der Applikation genommen.

Bestimmt die Farbe des Buttons.
Die möglichen Werte sind 'primary', 'warn', 'accent', 'default' / undefined. Über den LuxComponentsConfigService kann für alle LuxLinks applikationsweit eine Uppercase-Darstellung forciert werden.
