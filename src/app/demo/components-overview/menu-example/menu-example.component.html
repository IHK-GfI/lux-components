Über diese Property lassen sich CSS-Klassen für das ausgeklappte Menu setzen.
Um die Klassen zu beobachten, das Menu aufklappen und über die Developer-Tools (F12/CMD+ALT+I) das Menu auswählen.

LuxMenuItemComponent



Optionen - MenuItem #{{ i }}
Über dieses Flag kann der Eintrag als Kreis dargestellt werden.
Sollte nur in Zusammenhang mit einem Icon ohne Text genutzt werden, da dieser sonst das Layout sprengt. Dieses Flag stellt sicher, das dieser Eintrag immer außerhalb des Menus dargestellt wird, solange der Platz noch ausreicht.
Andere Einträge, die dieses Flag nicht gesetzt haben, werden dann zuerst ausgeblendet. Dieses Flag stellt ein ob das Label des MenuItems in der ausgeklappten Ansicht dargestellt werden soll oder nicht.
