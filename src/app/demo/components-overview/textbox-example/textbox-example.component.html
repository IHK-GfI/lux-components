Anwendungsbeispiele für die Farbthemen

Beispiel für Hinweise vom Typ "Success". Dazu folgende Propertys angeben:

luxTitle="Tipp"
luxIcon="lux-interface-validation-check-circle"
luxColor="green"

Beispiel für Hinweise vom Typ "Alert". Dazu folgende Propertys angeben:

luxTitle="Hinweis"
luxIcon="lux-interface-alert-warning-triangle"
luxColor="yellow"

Beispiel für Hinweise vom Typ "Warning". Dazu folgende Propertys angeben:

luxTitle="WARNUNG"
luxIcon="lux-interface-alert-warning-triangle"
luxColor="red"
